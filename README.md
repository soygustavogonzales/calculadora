Calculadora
===========
Esta es una calculadora basada en ajax con nodejs

1. Dependencias?
============
Luego de descargar el .zip

Express

```bash
> npm install express
```
o si clonaste el repositorio completo incluyendo el package.json original

```bash
> npm install
```

2. EJECUTANDO
=============

```bash
> $ node app.js

```

3. Visualizando
==============
En el navegador

localhost:4646
